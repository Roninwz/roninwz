# 快速上手

本节将介绍如何在项目中使用 roninwz-ui。

## 全局导入

```js
// 导入ui组件库
import RoninwzUI from "roninwz-ui";

import "roninwz-ui/lib/theme/index.css"; // 导入RoninwzUI样式

// 注册组件库
Vue.use(RoninwzUI);
```

## 特性

- 基于 `Vue` 开发的 UI 组件
- 使用 npm + webpack + babel 的工作流，支持 ES2015
- 提供友好的 API，可灵活的使用组件

## 浏览器支持

- 现代浏览器和 IE9 及以上
- [Electron](http://electron.atom.io/)
- [NW.js](http://nwjs.io)
